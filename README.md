# 哲思源流 — 西方古典哲學導覽

A minimalist, single-page HTML website introducing Western classical philosophy — its history, key thinkers, schools of thought, and the lineage of ideas from the Pre-Socratics to the Roman Stoics.

## Features

- **Hero section** — site introduction with a philosophical quote
- **何謂哲學 (What is Philosophy)** — four branches: metaphysics, ethics, epistemology, political philosophy
- **歷史脈絡 (Timeline)** — four classical eras: Pre-Socratic → Classical → Hellenistic → Roman
- **哲學家列傳 (Philosopher Cards)** — filterable grid of 12 philosophers, each with a detailed modal
- **思想傳承 (Lineage Diagram)** — SVG diagram showing teacher–student and influence relationships
- **哲學學派 (Schools)** — six major schools: Stoicism, Epicureanism, Platonism, Aristotelianism, Cynicism, Pyrrhonism
- **Quote marquee** — scrolling philosophical quotes
- Responsive design, smooth scroll, fade-in animations

## Usage

Open `index.html` directly in any modern web browser — no build step or server required.

```bash
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

## Stack

Pure HTML5 / CSS3 / Vanilla JavaScript — no frameworks or dependencies beyond Google Fonts.
